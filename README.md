# Security Labs

Kumpulan lab jaringan dan security yang saya kerjakan sebagai bagian dari persiapan karier di bidang Networking/Security Engineering. Setiap folder berisi topologi, konfigurasi, dan hasil pengujian dari lab yang bersangkutan.

## Daftar Lab

|Lab|Deskripsi|Skill yang Didemonstrasikan|
|-|-|-|
|[`home-lab-setup`](./home-lab-setup)|Environment lab pribadi (VirtualBox, Kali, Ubuntu) yang saling terhubung dalam satu jaringan|Hypervisor setup, network adapter configuration (NAT \& Host-only), Linux basics|
|[`nat-configuration`](./nat-configuration)|Konfigurasi NAT overload agar jaringan internal bisa akses internet lewat satu IP publik|NAT/PAT, routing dasar, Cisco IOS|
|[`vpn-site-to-site`](./vpn-site-to-site)|Konfigurasi VPN site-to-site menghubungkan dua jaringan berbeda lewat tunnel terenkripsi|IPsec VPN, crypto map, routing antar-site|

## Tools yang Digunakan

* Cisco Packet Tracer
* Oracle VirtualBox
* Kali Linux \& Ubuntu Desktop

## Tentang Repo Ini

Repo ini terus diperbarui seiring progres belajar saya mengikuti roadmap 6 bulan menuju posisi Security Engineer, mencakup fondasi networking, security fundamentals (TryHackMe, CompTIA Security+), praktik SOC analyst, dan cloud security (AWS).
